#My ExpenseTracker

A student-friendly expense tracking web app to record and review spending.

Live website: https://mystudexpenses.netlify.app/

What this project is for ?

My ExpenseTracker helps you:
Log expenses (amount + details)
Organize spending (commonly by category/date)
Review totals (e.g., daily/weekly/monthly trends)
To use it, you don’t need to install anything—just open the live site:

https://mystudexpenses.netlify.app/
